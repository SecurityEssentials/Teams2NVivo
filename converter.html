Transcript Converter

This page will convert a WebVtt transcript file from Microsoft Teams or Zoom into a formatted text file to import into NVivo

Choose a file from your computer using Choose file. This should be a text based file with a .vtt extension as downloaded from Teams or Zoom. You may want to enter a time shift value in seconds to add to each timestamp (typically -10 or so). Then click Convert and Download to create the converted .txt file
